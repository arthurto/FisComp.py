### Exercício 

Usando o interpretador de python da sua escolha plote os gráficos das seguintes funções em diferentes intervalos.

- $f(x) = sin(x)$ para $x\in(0,2\pi)$.
- $f(x) = e^{-x} cos(x)$ para $x\in(0,3)$.
- $f(x) = x^3 + x$ para $x\in(-1,1)$.
